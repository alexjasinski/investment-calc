# React Investment Calculator App

This is a simple React application for calculating investment results based on user input.

## 1. Features

- **User Input:** Collects information such as initial investment, annual investment, expected return, and investment duration.
- **Validation:** Ensures that the user enters a valid investment duration (greater than zero).
- **Results Display:** Presents the calculated investment results in a table.

## 2. Installation

### 2.1 Clone the Repository
git clone https://github.com/alexjasinski/investment-calc.git

### 2.2 Install Dependencies
cd your-repository
npm install

## 3. Run the Development Server
npm start
This will start the development server and open the app in your default web browser. If it doesn't open automatically, visit http://localhost:3000 in your browser.
